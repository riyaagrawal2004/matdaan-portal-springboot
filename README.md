# MatdaanPortal - Online Voting System

MatdaanPortal is a Spring Boot based online voting system that allows admins to manage voters, candidates, elections, and results, while voters can securely log in and cast their vote.

## Key Features

- Admin and voter login system
- Admin dashboard with voters, candidates, votes, and active election status
- Add, update, delete, and view voters
- Add, update, delete, and view candidates
- Set active election type
- Secure vote casting with duplicate vote prevention
- Live election result view
- Contact messages and forgot password requests stored in database
- Role-based access for admin pages
- Form validations for voter, candidate, election, and voting actions

## Tech Stack

- Java
- Spring Boot
- JSP
- JDBC
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

## Security & Validation

- Role-based access control
- Duplicate vote prevention
- Voter and candidate age validation
- Election type validation
- Empty form validation
- Confirmation popup for vote submit and delete actions

## How to Run

1. Clone the repository
2. Open the project in Eclipse or Spring Tool Suite
3. Create MySQL database named `matdaanportal`
4. Update database username and password in `application.properties`
5. Run the project as Spring Boot App
6. Open: `http://localhost:8080/`
